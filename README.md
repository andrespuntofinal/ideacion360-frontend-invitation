# Ideación 360 - Frontend

Frontend para el sistema de invitaciones de boda de Ideación 360.

## Tecnologías
- React
- Vite
- Tailwind CSS
- Framer Motion

## Configuración
1. Instalar dependencias: `npm install`
2. Iniciar servidor de desarrollo: `npm run dev`
3. Construir para producción: `npm run build`
